# C - Código Básico Parte 2

## Descrição

Este repositório contém exercícios e exemplos desenvolvidos em linguagem C com foco no aprendizado de conceitos intermediários da programação. O objetivo é reforçar a lógica de programação e o uso de estruturas fundamentais da linguagem.

---

## Conteúdos Abordados

Estruturas de repetição (`for`, `while`, `do...while`)

Estruturas de decisão (`if`, `else`, `switch`)

Funções

Vetores e matrizes

Manipulação de variáveis

Entrada e saída de dados

Exercícios práticos

---

## Requisitos

Para executar os programas, é necessário possuir um compilador C instalado, como:
GCC

MinGW (Windows)

Clang

Dev-C++

---

## Objetivo Educacional

Este projeto foi desenvolvido para praticar conceitos fundamentais da linguagem C e auxiliar estudantes iniciantes no desenvolvimento de habilidades de programação.
